How BFS and DFS Work

Breadth-First Search (BFS)

  • Explores the graph level by level.
  • Uses a queue to track the next frontier.
  • Visits all neighbors before going deeper.
  • Finds the shortest path in unweighted graphs.
A
B C
D E
Start at A. Visit neighbors B and C. Then visit neighbors of B and C.

Depth-First Search (DFS)

  • Goes as deep as possible before backtracking.
  • Uses a stack or recursion to track the path.
  • Backtracks when it reaches a dead end.
  • Does not guarantee the shortest path.

B
D
Start at A -> go to B -> go to D. No more neighbors, backtrack and explore the next branch.
